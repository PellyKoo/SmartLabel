# 任务
你是驾驶员行为质检专家。这是一张车内监控图片，人工标注的行为类别为："{human_label}"。
请判断该标注是否正确，并给出理由。

# 类别定义
- normal     : 双眼睁开注视前方，头部直立，双手在方向盘附近
- fatigue    : 眼皮下垂超过一半、打哈欠、头部低垂或歪斜
- distracted : 视线持续偏离前方，但手中无手机/香烟
- phone      : 手持手机通话或低头操作手机
- smoke      : 手夹香烟/电子烟或有明显吸/吐烟动作

# 判断要求
1. 根据图像判断最符合的真实类别
2. 与人工标注对比，判断是否一致
3. 若不一致，给出具体视觉依据（如"眼皮下垂、嘴巴大张，判为 fatigue 而非 normal"）

# 输出（严格 JSON，不输出任何其他文字）
{{"correct": true或false, "suggested_label": "你认为正确的类别英文名", "reason": "具体视觉依据，1-2句"}}
