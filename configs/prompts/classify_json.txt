# 任务
你是驾驶员行为分析专家。观察图中驾驶员，判断其当前行为状态。

# 类别定义（严格按以下视觉特征判断）
- normal     : 双眼睁开、注视前方；头部姿态直立；双手自然握持方向盘或置于附近
- fatigue    : 眼睛闭合或眼皮明显下垂超过一半、打哈欠张嘴、头部下垂或左右歪斜、频繁缓慢眨眼
- distracted : 视线持续偏离前方（向左/右/后方凝视）但手中无手机/香烟；例如整理头发、与乘客说话
- phone      : 手持手机/平板贴近耳边通话，或低头盯着手机屏幕操作，或将手机放在腿上看
- smoke      : 手指间夹着香烟或电子烟，或嘴部出现明显吐烟/吸烟口型

# 易混淆提示
- 打哈欠头会短暂低垂但视线会回正 → fatigue；视线长时间离开前方 → distracted
- 手持设备 → phone；仅扭头看乘客无设备 → distracted
- 戴墨镜但面朝前方 → normal（不因看不清眼睛就判 fatigue）
- 低头看仪表盘或导航短暂 → normal；持续低头看手机屏 → phone
- 烟雾 / 夹烟动作 → smoke，不归入 distracted

# 判断步骤
1. 一句话描述关键线索：眼睛状态、头部朝向、双手持有物
2. 对照类别定义选出最匹配的一个标签

可选类别：{categories}
