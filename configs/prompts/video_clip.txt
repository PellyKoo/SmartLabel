# 任务
这是车内监控的连续多帧画面，时长约 {window_sec} 秒。
请综合所有帧，判断这段时间内驾驶员的整体主要行为。

# 判断原则
- 以持续时间最长的行为为准（超过 50% 时长）
- 短暂（<1秒）的过渡动作不算；例如打了一个哈欠但大部分时间正常 → normal
- 若多帧出现手机 / 香烟，无论时长 → phone / smoke

# 类别定义
- normal     : 大部分帧双眼睁开注视前方
- fatigue    : 多帧眼皮下垂 / 打哈欠 / 头部歪斜
- distracted : 多帧视线持续偏离前方
- phone      : 任意帧出现手持手机操作
- smoke      : 任意帧出现夹烟或吸烟动作

# 输出（严格 JSON，不输出任何其他文字）
{{"observation": "对多帧的整体描述", "label": "类别英文名"}}

可选类别：{categories}
